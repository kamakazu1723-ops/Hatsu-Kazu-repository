# 初音ちゃん画像セットアップガイド

## 画像の配置方法

プロフィールページで初音ちゃんの写真を表示するには、以下の手順を実行してください：

### 1. 画像ファイルを保存

初音ちゃんの写真を `public/hatsu-chan.jpg` として保存してください。

**ファイルパス:**
```
Frontend/
  public/
    hatsu-chan.jpg    ← ここに画像を配置
    favicon.svg
    icons.svg
```

### 2. 画像の推奨仕様

- **ファイル形式**: JPG, PNG, WebP
- **推奨サイズ**: 600x800px 以上
- **ファイル名**: `hatsu-chan.jpg` (他の名前を使う場合は App.jsx も更新が必要)

### 3. 動作確認

1. 画像をコピーして `public/` フォルダに配置
2. ブラウザをリフレッシュ
3. ナビゲーションバーの「👤 プロフィール」をクリック
4. 初音ちゃんの素敵な写真が表示されます！

## ページの機能

✨ **ホームページ**
- 浮いしているハートのアニメーション
- 「初音ちゃんへの愛」カウンター
- かわいい統計情報カード

👤 **プロフィールページ**
- 初音ちゃんの写真（グロー効果付き）
- プロフィール情報
- 応援ボタン

## カスタマイズ

### 画像ファイル名を変更したい場合

`src/App.jsx` の以下の部分を編集してください：

```jsx
<img 
  src="/hatsu-chan.jpg"    ← ここを変更
  alt="初音ちゃん" 
  className="profile-image"
/>
```

例：`src="/my-image.png"` に変更
